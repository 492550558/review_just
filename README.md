# 每日复盘小助手 (Daily Review App)

## 1. 项目概述

一款面用来记录每日任务和复盘的应用，主要用于帮助用户进行每日任务的复盘与反思。用户可以轻松记录每天完成的事项并通过月视图直观地回顾整月的进度与状态。提供统计分析功能，能够从数据中了解自己的习惯趋势、任务完成率从而更有针对性地调整计划、提升效率。

### 主要功能
- **每日清单**：创建、管理复盘记录每日任务，支持拖拽排序、状态标记（完成/失败）及优先级设置。
- **日历视图**：提供月度日历视图，直观展示每日复盘情况，支持快速跳转至指定日期。
- **数据统计**：通过折线图和饼状图展示任务完成趋势与状态占比，支持按周、按月维度分析。
- **本地存储**：数据存储在本地，保护用户隐私，无需联网即可使用核心功能。

### 目标用户
- 需要进行时间管理和任务追踪的学生或职场人士。
- 习惯每日反思、追求个人成长的终身学习者。

### 技术栈
- **核心框架**：微信小程序原生开发 (WXML, WXSS, JavaScript, JSON)
- **图表渲染**：Canvas 2D API
- **数据存储**：wx.setStorageSync / wx.getStorageSync

## 3. 功能特性详细说明

### 主页
- **任务管理**：点击底部 "+" 号添加新任务，支持输入内容和选择优先级（高/中/低）。
- **交互操作**：
  - **长按拖拽**：长按任务项可上下拖拽排序。
  - **左滑删除**：向左滑动任务项可进行删除操作。
  - **状态切换**：点击左侧圆圈标记完成/失败/进行中。
- **顶部日历**：显示当前周，点击日期切换查看历史记录。

### 日历 
- **月视图**：展示整月日历，标记有记录的日期。
- **快捷跳转**：点击任意日期可跳转回主页查看当日详情。
- **返回本月**：在浏览历史月份时，右下角会出现悬浮按钮，一键返回当前月份。

### 统计
- **趋势分析**：
  - **按周统计**：展示本周（周一至周日）的任务完成数量折线图。
  - **按月统计**：展示本年度（1月至12月）的任务完成趋势。
- **状态占比**：饼状图展示任务的完成率、失败率等分布情况。
- **数据概览**：顶部卡片显示总任务数、完成数及成功率。

## 4. 配置说明

### AppID 配置
在 `project.config.json` 文件中，修改 `appid` 字段为您自己的小程序 AppID：
```json
{
  "appid": "wx12345",
  ...
}
```

### 颜色与样式
- **主题色**：主要使用了微信绿 (`#07c160`) 作为主色调。
- **导航栏颜色**：在各页面的 `.json` 文件中配置 `navigationBarBackgroundColor`。
